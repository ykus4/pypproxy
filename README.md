# paxy

MITM HTTP/HTTPS proxy for inspecting and modifying traffic.

[![CI](https://github.com/ykus4/paxy/actions/workflows/ci.yml/badge.svg)](https://github.com/ykus4/paxy/actions/workflows/ci.yml)
[![PyPI](https://img.shields.io/pypi/v/paxy)](https://pypi.org/project/paxy/)

```bash
pip install paxy
paxy          # GUI mode  →  http://localhost:8081
paxy --mode cui  # terminal UI
```

See the **[docs](https://ykus4.github.io/paxy/)** for setup, CA installation, and feature guides.
